# 《当你不在的世界》微信小游戏原型

这是一个可运行的微信小游戏原型，实现了你给出的策划案中的核心结构：

- 9 个关卡（每关一个情感主题）
- 黑白到彩色的全局色彩恢复系统
- 记忆碎片收集与拼图地图
- 角色选择（男孩 / 女孩）
- 基础平台跳跃 + 逐关机制演示（反向、迷雾、二段跳、节奏、镜像、暴雨、综合）
- 本地存档（关卡进度、碎片、星星、色彩进度）
- 结局页与微信分享调用

## 目录

- `game.js`：小游戏入口
- `js/main.js`：核心游戏逻辑

## 在微信开发者工具中运行

1. 打开微信开发者工具。
2. 导入本项目目录 `/workspace/wxdemo`。
3. 选择 **小游戏** 项目类型。
4. 若工具提示缺少配置，可新建默认 `project.config.json`（开发者工具会自动生成）。
5. 点击运行。

## 玩法

- 主菜单选择角色后进入拼图地图。
- 点击已解锁关卡方块开始挑战。
- 左侧区域触控移动，右侧区域触控跳跃。
- 第 5 关后收集碎片可解锁二段跳。
- 每关 25 秒内通关可获时间之星。

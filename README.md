# 《当你不在的世界》微信小游戏（完善版原型）

基于策划案实现的微信小游戏可玩原型，已覆盖：情感叙事主流程、9关卡结构、色彩恢复、拼图收集、双角色差异文本、设置与存档。

## 已实现能力

- 入口与配置：`game.js` + `game.json`
- 主菜单：角色选择（男孩/女孩）
- 设置面板：音乐开关、音效开关、摇杆模式切换、清空进度
- 拼图地图：9宫格选关、关卡解锁、碎片标记、星星标记
- 9个关卡机制演示：
  - 基础跳跃 / 距离沟壑 / 反向控制 / 迷雾视野
  - 二段跳解锁 / 节奏平台 / 镜像布局 / 暴雨干扰 / 综合关
- 系统：
  - 色彩恢复进度（0% → 100%）
  - 每关记忆碎片收集
  - 时间之星判定（关卡阈值）
  - 暂停菜单（继续 / 重开 / 返回地图）
  - 本地存档（进度、星星、碎片、设置）
- 结局：重逢页 + 分享文案调用

## 目录

- `game.js`：小游戏入口
- `game.json`：小游戏配置
- `js/main.js`：核心逻辑（场景、玩法、UI、存档）

## 运行方式（微信开发者工具）

1. 打开微信开发者工具
2. 导入本项目目录 `/workspace/wxdemo`
3. 选择 **小游戏** 类型
4. 如需，自动生成 `project.config.json`
5. 点击运行

## 控制说明

- 游戏内：
  - 屏幕左侧区域：向左移动
  - 屏幕中左区域：向右移动
  - 屏幕右侧区域：跳跃（解锁后可二段跳）
- 左上显示关卡、色彩和计时；右上可暂停
